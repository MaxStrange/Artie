# Artie Workbench Contribution Guide

This document provides guidelines and best practices for contributing code
to Artie Workbench.

## Overview of Artie Workbench

Artie Workbench is a graphical user interface (GUI) application written in
PyQt. Its purpose is to provide a nice user interface for administering
Arties.
